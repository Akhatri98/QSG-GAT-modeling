# TF-IDF Model

Pipeline for prediciton open-close movements in stoc prices from overnight offering news. Uses TF-IDF features on headlines and body texts, with manual scalars. Trained with ridge and logistic regression.

## Install

```bash
pip install pandas numpy scipy scikit-learn matplotlib
```

## Run Order

```bash
py src/build_index.py
py src/extract_features.py
py src/train_evaluate.py
py src/visualize.py
```

## Data

Copy tsv files into the raw_data folder, or change the path in utils.py to point towards your dir. Files generated by visualize.py are put into the notebook folder, all other files generated will be in the clean_data folder.
